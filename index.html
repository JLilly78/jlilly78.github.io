C++ music player that I made

(A work in progress). Utilizes the PortAudio library with ASIO drivers. I wrote the logic for the callback function to feed in wav file data to the output buffer. It makes use of a PortAudio library ring buffer to cache the audio data within a thread function and then sends that to the callback function to be sent to the ASIO output buffer. It makes use of global states to know when to pause/stop/resume/play. The program itself uses WIN32 to run as a window. It's capable of WAV and MP3 playback. It displays the track name and number for now.
